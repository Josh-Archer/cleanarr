# Security Review Gate

Verify before first public push:

- No secrets, local databases, kubeconfigs, or sealed secrets are tracked.
- No internal domains, personal usernames, or private ntfy topics remain.
- Example manifests use placeholders only.
- Workflows do not depend on private secrets or private reusable workflows.
- Defaults are generic and safe for public use.
- Generated bytecode and other transient artifacts are not tracked.
